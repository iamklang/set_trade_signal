ส่วนที่ 1: ภาพรวมระบบ — มันทำอะไร?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ระบบนี้ทำ 3 อย่าง:
  1. บอก "เทรนด์" — ราคาอยู่เหนือ/ใต้เส้นส้ม (20 EMA)
  2. บอก "จุดเข้า" — ซื้อตอนราคาย่อลงมาแตะเส้นส้มแล้วเด้ง (dip-buy)
  3. บอก "จุดออก + ขนาดไม้" — Stop / Target / Position size

สิ่งที่ script ทำไม่ได้ (ต้องใช้สมองคนเอง):
  - อ่าน macro (สงคราม/สันติ, น้ำมัน, เฟด)
  - ดู sector rotation (ใครแบกตลาด)
  - วิเคราะห์งบ (beat/miss, sell-on-fact, Selling Climax)
  - เลือก DW (delta, IV, expiry)

ส่วนที่ 2: วิธีอ่านหน้าจอ — สิ่งที่เห็นบนชาร์ตหมายความว่าอะไร
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[แท่งเทียน]
  สีเขียว = ราคาอยู่เหนือเส้นส้ม → เทรนด์ขึ้น → หาจังหวะซื้อตอนย่อ
  สีแดง   = ราคาอยู่ใต้เส้นส้ม → เทรนด์ลง → ห้ามซื้อ (เล่นเด้งอย่างเดียว)

[เส้น]
  เส้นส้ม (20 EMA)   = เส้นเทรนด์หลัก — ถ้าราคาอยู่เหนือ = ขาขึ้น
  เส้นเทา (200 SMA)  = แนวรับ/ต้านโครงสร้างใหญ่
  วงกลมแดง           = แนวต้าน (Resistance) จาก pivot
  วงกลมเขียว         = แนวรับ (Support) จาก pivot
  เส้นประแดง          = Stop Loss (ใต้ swing low)
  เส้นจุดเขียว        = Target 1 (1R) และ Target 2 (1.5R)

[ป้าย BUY / SELL]
  ป้ายเขียว "BUY (dip)" ใต้แท่ง = สัญญาณซื้อ (ย่อแตะเส้นส้ม + แท่งเขียวยืนยัน)
    → แสดง Entry / Stop / TP ให้เลย
  ป้ายส้ม "BUY (dip) + earnings near" = ซื้อได้แต่ระวัง งบใกล้ออก → IV crush
  ป้ายแดง "SELL" เหนือแท่ง = ออกจากสถานะ
    → "trend break" = ราคาหลุดเส้นส้ม
    → "resistance"  = ราคาชนแนวต้านแล้วปิดแดง

[Dashboard มุมขวาบน]
  ACTION     = สรุปว่าตอนนี้ควรทำอะไร (ดูตรงนี้ก่อนเลย)
    "BUY"         = มีสัญญาณซื้อ
    "SELL"        = มีสัญญาณขาย
    "HOLD"        = ถือรอย่อ (เทรนด์ขึ้นแต่ยังไม่ย่อ)
    "STAND ASIDE" = ห้ามเล่น (เทรนด์ลง)
  Trend      = UP / DOWN
  Px vs EMA  = ราคาห่างจากเส้นส้มกี่ %
  Support/Res = แนวรับ/แนวต้านจาก pivot
  Buy / Stop = ราคาเข้า / จุด stop
  1R / 1.5R TP = เป้า take profit
  Size       = ขนาดไม้ (คำนวณจาก equity x risk%)
  Win rate   = สถิติย้อนหลังบนชาร์ต (simulation)
  Expectancy = ค่าเฉลี่ย R ต่อเทรด (บวก = มี edge)

[สัญลักษณ์ Advanced (ต้องปิด Simple Mode)]
  SC (สีเลือดหมู) = Selling Climax — ขายทิ้งหนักแต่ราคาไม่ลง → โอกาสสะสม
  วงกลมฟ้า       = Accumulation — volume สูงแต่ราคาไม่ตก
  กากบาทแดง      = Fake breakout — ราคาแลบเหนือแนวต้านแต่ปิดแดง → ห้ามไล่ซื้อ
  เพชรม่วง       = วันประกาศงบ (ย้อนหลัง — เฉพาะแท่งที่ EPS เปลี่ยน)

ส่วนที่ 3: ขั้นตอนการเทรด — ทำตามนี้ทีละ step
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: อ่าน Macro (ทำก่อนเปิดชาร์ต)
--------------------------------------
  ถามตัวเองว่า:
  - สงครามหรือสันติภาพ?
  - น้ำมันขึ้นหรือลง?
  - Nasdaq/NDX แข็งหรืออ่อน?
  - มี event สำคัญ (เฟด, กนง., งบ) เมื่อไหร่?

  ถ้าน้ำมันลง → ซื้อกลุ่ม anti-commodity (ปิโตรเคมี, ไฟฟ้า, สายการบิน)
  ถ้าน้ำมันขึ้น → ซื้อกลุ่ม E&P (PTTEP, IRPC)
  ถ้า NDX แข็ง → ซื้อกลุ่มชิป (DELTA, HANA, CCET)


Step 2: ดูเทรนด์ SET Index
---------------------------
  เปิดชาร์ต SET (Daily) ใส่ indicator "SET DW Swing"
  ดู ACTION box:
  - ถ้า "UP (buy dips)" → หาจังหวะซื้อตอนย่อ
  - ถ้า "DOWN" → ห้ามซื้อยาว เล่นเด้งสั้นอย่างเดียว


Step 3: หาหุ้นที่จะเล่น
-------------------------
  เลือกหุ้นจากกลุ่มที่ macro สนับสนุน (Step 1) ที่กำลังย่อลงมา
  เปิดชาร์ต Daily ใส่ indicator แล้วดู:
  - แท่งเทียนสีเขียว? (เทรนด์ขึ้น)
  - มี BUY label หรือยัง?
  - ถ้ายังไม่มี → ดู Px vs EMA ว่าใกล้เส้นส้มแค่ไหน → รอย่อ


Step 4: ตัดสินใจเข้า — เมื่อ BUY ขึ้น
---------------------------------------
  เมื่อเห็นป้าย BUY (เขียว) บนชาร์ต:
  1. ดูราคา Entry, Stop, TP จากป้าย
  2. คำนวณ R: ถ้า entry 100, stop 95 → 1R = 5 บาท, target 1R = 105, 1.5R = 107.5
  3. ดู Size จาก dashboard → กี่หุ้น
  4. ตรวจสอบ:
     - งบใกล้ออกไหม? (ดู Earnings ใน dashboard)
     - Volume ปกติไหม? (ไม่มี SC หรือ fake breakout)
     - Macro ยัง support ไหม?
  5. ถ้าทุกอย่าง OK → เข้าซื้อ


Step 5: จัดการ position
------------------------
  - วาง Stop Loss ที่ระดับจาก script (เส้นประแดง)
  - เมื่อถึง Target 1 (1R) → ขายครึ่งหนึ่ง, ขยับ stop ขึ้นมาที่ราคาเข้า (breakeven)
  - เมื่อถึง Target 2 (1.5R) → ขายที่เหลือ
  - ถ้าเห็น SELL label → ออกทันที ไม่ต้องรอ target


Step 6: เลือก DW (ถ้าเทรด DW แทนหุ้น)
----------------------------------------
  ดูจาก warrant card ของ broker:
  - Delta 0.4–0.6 (สมดุลระหว่าง tracking กับ leverage)
  - Effective gearing = gearing x delta → ยิ่งสูง leverage ยิ่งเยอะ
  - IV ต่ำกว่าตัวอื่นบนหุ้นเดียวกัน = ถูกกว่า
  - Expiry ไกล (ไม่น้อยกว่า 2–3 เดือน) → ลด theta decay
  - ระวัง IV crush ถ้างบใกล้ออก → ห้ามซื้อ DW ก่อนงบ

ส่วนที่ 4: ตัวอย่างการเทรดจริง (สมมุติ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ตัวอย่าง 1: ซื้อ DELTA ตอนย่อ (กรณีปกติ)
-------------------------------------------

  Macro: สันติภาพ, น้ำมันลง, NDX แข็ง → ดี sector ชิป
  SET Index: เหนือ 20-EMA → เทรนด์ขึ้น

  เปิดชาร์ต DELTA Daily ใส่ SET DW Swing:
  - แท่งเทียน: สีเขียว (เทรนด์ขึ้น)
  - ราคา 180 บาท, EMA อยู่ที่ 175
  - DELTA ย่อลงมาแตะ 176 (ใกล้ EMA) แล้ววันนี้ปิดเขียว 180
  - Script ขึ้น BUY label:

    ┌────────────────────┐
    │  BUY (dip)         │
    │  Entry  180.00     │
    │  Stop   172.00     │
    │  TP     188.00     │
    └────────────────────┘

  คำนวณ:
  - 1R = 180 - 172 = 8 บาท
  - Target 1 (1R) = 180 + 8 = 188
  - Target 2 (1.5R) = 180 + 12 = 192
  - ขนาดไม้: equity 1,000,000 x 1% = 10,000 / 8 = 1,250 หุ้น

  ซื้อ 1,250 หุ้น @ 180
  Stop: 172 (ถ้าหลุดขาย ขาดทุน 10,000 = 1% ของพอร์ต)

  วันต่อมา:
  - ราคาขึ้นถึง 188 → ขาย 625 หุ้น (ครึ่งหนึ่ง) กำไร 8x625 = 5,000
  - ขยับ stop ขึ้นมาที่ 180 (breakeven)
  - ราคาขึ้นต่อถึง 192 → ขายอีก 625 หุ้น กำไร 12x625 = 7,500
  - รวมกำไร: 12,500 บาท (+1.25% ของพอร์ต)


ตัวอย่าง 2: ซื้อ DW บน DELTA
-------------------------------

  จากตัวอย่าง 1 แต่ใช้ DW แทนหุ้น

  ดู warrant card:
  - DELTA13C2609A: delta 0.55, gearing 5.0x, IV 28%, expiry ก.ย. 2026
  - DELTA08C2608B: delta 0.80, gearing 3.2x, IV 32%, expiry ส.ค. 2026

  เลือก DELTA13C2609A เพราะ:
  - Effective gearing = 5.0 x 0.55 = 2.75x (leverage จริง)
  - IV 28% < 32% → ถูกกว่า
  - Expiry ก.ย. → ไกลพอ

  DELTA ขึ้น 4.4% (180→188) → DW ขึ้นประมาณ 4.4% x 2.75 = ~12%
  (จริงจะไม่ตรงเป๊ะเพราะ IV เปลี่ยน + theta กิน แต่ก็ขยายกำไรชัดเจน)


ตัวอย่าง 3: เห็น SELL — ออกจากสถานะ
--------------------------------------

  ถือ GULF อยู่ entry 52, stop 49
  วันนี้ GULF ขึ้นไป 56 ชนแนวต้าน แล้วปิดแดงที่ 55
  Script ขึ้น SELL label "resistance"

    ┌────────────────────┐
    │  SELL              │
    │  (resistance)      │
    │  55.00             │
    └────────────────────┘

  ขายที่ 55 → กำไร 3 บาท/หุ้น = 1R (stop ห่าง 3)
  ไม่ต้องเสียดาย — ขายที่แนวต้านแล้วรอซื้อรอบใหม่ตอนย่อ


ตัวอย่าง 4: งบใกล้ออก → ระวัง IV Crush
-----------------------------------------

  CPF จะประกาศงบใน 3 วัน
  เปิดชาร์ต CPF → เห็น BUY label สีส้ม พร้อมเตือน "earnings near"

    ┌────────────────────┐
    │  BUY (dip)         │
    │  !! earnings near  │
    │  Entry  21.80      │
    │  Stop   20.50      │
    │  TP     23.10      │
    └────────────────────┘

  ทางเลือก:
  a) ซื้อหุ้น CPF ตรง (ไม่โดน IV crush) แล้ว stop structural
  b) รองบออกก่อน → ถ้างบดี ราคา gap up → ตามซื้อ DW หลังงบ (IV ลดแล้ว ถูกกว่า)
  c) ถ้าเสี่ยงไม่ไหว → อย่าถือข้ามงบ

  *** ห้ามซื้อ DW ก่อนงบ → IV ถูก mark up → งบออกแล้ว IV ยุบ → แม้ทิศทางถูก ก็ขาดทุน ***


ตัวอย่าง 5: Selling Climax — ซื้อหลังโดนเทงบเสีย
---------------------------------------------------

  HANA ประกาศงบ miss → ราคาร่วงจาก 45 ลง 38 ใน 2 วัน พร้อม volume พุ่ง
  Script (Advanced mode) ขึ้น "SC" (Selling Climax)
  ราคาลงมาถึงฐานเก่าที่ 37–38 → volume เริ่มแห้ง

  ถ้า HANA ยังกำไร/จ่ายปันผล → อาจซื้อที่ฐาน 38, stop 35
  1R = 3, target = กลับ EMA ที่ 43 = 5 บาท = 1.67R
  ขนาดไม้: 1,000,000 x 1% / 3 = 3,333 หุ้น


ตัวอย่าง 6: STAND ASIDE — ห้ามเล่น
-------------------------------------

  เปิดชาร์ต SET Index → แท่งสีแดง, Action "STAND ASIDE"
  ราคาอยู่ใต้เส้นส้ม → เทรนด์ลง

  สิ่งที่ควรทำ: ไม่ซื้อ, ถือเงินสด, รอ SET กลับเหนือ EMA
  สิ่งที่ห้ามทำ: ไปไล่ซื้อตอนตลาดร่วง "เห็นว่าถูก" → ดอย

ส่วนที่ 5: กฎเหล็ก (สรุปจากทั้ง backtest + method)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. ซื้อตอนย่อ ไม่ไล่ราคา → "ไล่ = ดอย"
  2. รอแท่งเขียวยืนยันก่อน → อย่าซื้อตอนยังร่วง
  3. Stop = ระดับโครงสร้าง → ถ้าหลุด ออก ไม่ต่อรอง
  4. Size = 1% ของพอร์ต → ขาดทุน 1 เทรดไม่เจ็บ, ถูก 10 เทรดจาก 100 ก็อยู่ได้
  5. ห้าม average down → โดยเฉพาะ DW ที่มี theta + IV decay
  6. ขายที่แนวต้าน → อย่าโลภรอสูงกว่า
  7. ห้ามซื้อ DW ก่อนงบ → IV crush ทำให้ทิศถูกก็ขาดทุน
  8. Macro มาก่อนชาร์ต → งบดีแต่สงครามปะทุก็โดนเท
  9. ขนาดไม้สำคัญกว่าจุดเข้า → backtest พิสูจน์แล้ว (DD 43%→6.5%)
  10. ชาร์ตอย่างเดียวไม่พอ → backtest 10 ปีเสมอตัว ต้องใช้ macro+sector+งบ ร่วม

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ทดสอบ Python 25 หุ้น SET / 10 ปี / ค่าคอม 0.157%:

  แบบกลไกล้วน (ไม่มี macro/งบ):
  - Fixed size: -19%, PF 0.91, Max DD 43% → ขาดทุน
  - Risk 1%:    -5%,  PF 0.80, Max DD 6.5% → ขาดทุนน้อยลงเพราะ sizing ดี

  ปิด EMA exit (ให้ stop + target ทำงานเอง):
  - PF 1.09, +1%, Max DD 1.8% → พอเสมอตัว

  เพิ่ม filter (RSI>=55, 200-SMA rising, ADX>=20, breakeven after T1):
  - Win rate 61%, PF 1.11, avgR +0.03

  สรุป: กลไกชาร์ตอย่างเดียว "เสมอตัว" เท่านั้น
  Edge จริงต้องมาจาก macro + sector + งบ + วินัย sizing
  ระบบนี้เป็น "โครงสร้าง" ที่ดี แต่ไม่ใช่ money machine อัตโนมัติ

ส่วนที่ 7: Settings ที่ต้องตั้งเอง (ใน TradingView)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ค่าที่ต้องใส่ทุกตัว:
  - Account equity: พอร์ตจริง (เช่น 1,000,000)
  - Risk %: 1% (default, ไม่เกิน 2%)

  ค่าที่ต้องใส่ถ้าเทรด DW:
  - เปิด "This is a DW" → ใส่ gearing และ delta จาก warrant card

  ค่าที่ต้องใส่ช่วงงบ:
  - Next earnings date → ใส่วันประกาศงบจากปฏิทิน broker

  ค่า default ที่ backtest แล้ว (แนะนำไม่ต้องเปลี่ยน):
  - EMA 20, RSI min 55, ADX min 20, 200-SMA rising ON
  - Target 1R / 1.5R, breakeven after T1 ON
  - Trend-break exit OFF (ทำให้ whipsaw)

  ปรับเพิ่มตามหุ้น (ถ้าอยากละเอียด):
  - DELTA: RSI 60, prior-bar-red ON, stop lookback 12
  - RCL:   RSI 65, volume-confirm ON, stop lookback 12
  - HANA:  RSI 60, ADX 20, target 1R/1.5R, stop 10-12

ส่วนที่ 8: Flowchart สรุป — ตัดสินใจในหัว
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Macro ดีไหม?
  ├─ ไม่ → STAND ASIDE
  └─ ใช่ → SET เหนือ EMA ไหม?
           ├─ ไม่ → รอ (เล่นเด้งสั้นได้)
           └─ ใช่ → หุ้นที่ดูอยู่เหนือ EMA ไหม?
                    ├─ ไม่ → ข้ามไปตัวอื่น
                    └─ ใช่ → ย่อมาแตะ EMA แล้วเด้งไหม? (BUY label?)
                             ├─ ไม่ → รอย่อ
                             └─ ใช่ → งบใกล้ออกไหม?
                                      ├─ ใช่ → ซื้อหุ้นตรง หรือรองบก่อน
                                      └─ ไม่ → ซื้อ + วาง stop + target
                                               → ถึง T1: ขายครึ่ง, stop breakeven
                                               → ถึง T2: ขายหมด
                                               → SELL label: ออกทันที
